## ⚡ How I build

```
No framework. No bundler. No build step.
Just HTML, CSS, and JavaScript — straight to GitHub Pages.
```

- **Offline-first** — service workers, cache strategies
- **Multilingual** — 12 languages baked in, no external libs
- **PWA** — installable on Android & iOS
- **Self-hosted** — GitHub Pages + cedracine.com
