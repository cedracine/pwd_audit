PASSWORD AUDIT — PACKAGE
========================

Contenu :
  pwd_audit_guide.html  →  Ouvre ce fichier dans ton navigateur pour lire le guide
  pwd_audit_gui.py      →  Le script Python à lancer après avoir lu le guide

Démarrage rapide :
  1. Ouvre pwd_audit_guide.html dans Chrome/Firefox
  2. Lis la section "Installer Python"
  3. Lance : python pwd_audit_gui.py  (Windows)
             python3 pwd_audit_gui.py (Mac/Linux/Raspi)

Aucune donnée n'est envoyée sur internet (sauf vérification domaines morts, optionnelle).
